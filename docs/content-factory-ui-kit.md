# UI-кит раздела «Контент-Завод»

## Цель
Обеспечить единый визуальный стиль раздела «Контент-Завод» и согласовать его с общей дизайн-системой приложения.

## Базовые принципы
- Единый ритм отступов: 4 / 8 / 12 / 16 / 24 / 32.
- Четкая иерархия заголовков: H1, H2, H3.
- Контрастные состояния элементов: normal, hover, active, disabled.
- Простые и понятные тексты без технических терминов.
- Адаптивность от 320px до 4K за счет fluid-отступов и адаптивной сетки.

## Компоненты UI-кита
- `CfButtonMd`: единая кнопка среднего размера для главных и вторичных действий.
- `CfSection`: стандартный контейнер функционального блока.
- `CfH1`, `CfH2`, `CfH3`: стандартизированные заголовки.
- `cfStyles`: набор общих классов для полей, подсказок, вкладок и сетки.

## Типографика
- H1: ключевой заголовок страницы.
- H2: заголовок крупного блока.
- H3: заголовок локального блока или карточки.
- Body: основной текст интерфейса.
- Hint: поясняющие подписи и короткие подсказки.

## Цветовая система
- Основной акцент: `primary`.
- Фоновые поверхности: `card`, `secondary`.
- Служебные состояния: `destructive`, `muted-foreground`.
- Все цвета привязаны к токенам темы, без локальных «случайных» цветов.

## Адаптивная сетка
- Мобильные: 320–639px — один столбец, компактные отступы.
- Планшеты: 640–1023px — один/два столбца в зависимости от блока.
- Десктоп: 1024px+ — два столбца в блоке создания контента.
- Wide/4K: ограничение ширины контента и увеличение внутренних отступов для удобства чтения.

## A/B тестирование интерфейса
- Назначение варианта: `A` или `B` на уровне сессии через `localStorage`.
- События: открытие страницы, переключение вкладок, запуск генерации, успешное завершение, ошибка.
- Метрики:
  - Вовлеченность = доля сессий с запуском генерации.
  - Успешность задачи = доля завершенных генераций.
  - Время выполнения = среднее время от клика «Создать» до статуса «готово».
- Текущий прогресс по выборке отображается в интерфейсе «История».

## Проверка качества
- Ручная проверка сценариев: создание по ссылке и по теме.
- Проверка записи голоса, очистки записи и отображения результата.
- Проверка состояний: загрузка, ошибка, пустая история, готовый контент.
- Проверка на экранах: 320px, 768px, 1024px, 1440px, 2560px, 3840px.
